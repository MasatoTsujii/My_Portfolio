## 概要
OffSeason2019に設計した基板データである。
設計には **Eagle** を使用した。
閲覧用として、各プロジェクト内に回路図 (`Schematic.pdf`) および基板図 (`PCB.pdf`) を同梱している。

## ディレクトリ構成と基板説明

### 1. EmergencyStop_Remote
**概要:** ロボットの遠隔非常停止。
* **役割:** 大会のルールで自動制御を行う際は遠隔非常停止をつける必要があり開発。
* **特徴:** Arduino MINIが処理を行い、スイッチが押されるとリレーが動作し電源が切れる。
* **使用結果:** 3月にコロナの影響で開発ができなくなり、手動ロボットとなったため使用しなかった。

### 2. MD_Middle
**概要:** チームが多くモータドライバが足りなかった為自作したモータドライバ。
* **役割:** ロボットの機構で使用するモータのトルクや回転速度を制御する基板。
* **特徴:** これまではリレーとFETを用いたMDを使用していたが、FETのみで構成されている。
* **使用結果:** パターンが電流に対して狭すぎて、大会前日の運用中に**燃えた**。

### 3. NucleoMB_Middle
**概要:** ロボットのメイン制御ボード（マザーボード）
* **役割:** メインマイコンである **Nucleo-64 (STM32F446RE)** を搭載し、各センサやアクチュエータへのアクセスピンを使いやすい配置に引き出すための基板。
* **特徴:** 配線の煩雑さを解消し、メンテナンス性を向上させるために設計。
* **使用結果:** 問題なく動作したが自動制御を行わなかった為、汎用的なものでよかった。
