## ハイライト (Highlights)

### 1. 通信環境と操作性の課題解決 (Season2022)
**課題:** 10基の砲塔を個別制御するためのボタン不足と、現地会場での電波干渉（2.4GHz帯）のリスク。
**解決策:**
* **専用コントローラーの自作:** 市販品では不可能なボタン配置と機能を実装。
* **920MHz帯の採用:** 混信に強いIM920sLモジュールを採用し、通信プロトコルを独自に実装・改良。大会での通信エラーゼロを達成しました。
* [詳細はこちら](./Season2022)

### 2. ダイレクトティーチングによる演出強化 (Season2020)
**課題:** コロナ禍による活動制限で、複雑なモーションプログラムを組む時間が不足。
**解決策:**
* **直感的な教示システム:** ロボットのアームを手で動かして軌道を記録・再生する機能を実装。
* **成果:** プログラムが書けないメンバーでも演出作成が可能になり、開発効率を向上。「本田技研工業特別賞」を受賞しました。
* [詳細はこちら](./Season2020)
