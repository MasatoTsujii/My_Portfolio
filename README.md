## ハイライト

### 1. 通信環境と操作性の課題解決 (Season2022)
**課題:** 10基の砲塔を個別制御するためのボタン不足と、現地会場での電波干渉（2.4GHz帯）のリスク。

**解決策:**
* **専用コントローラーの自作:** 市販品では不可能なボタン配置と機能を実装。
* **920MHz帯の採用:** 混信に強いIM920sLモジュールを採用し、通信プロトコルを独自に実装・改良。大会での通信エラーはなかった。
* [詳細はこちら](./Season2022)

### 2. ダイレクトティーチングの実装 (Season2020)
**課題:** コロナ禍による活動制限で、複雑なモーションプログラムを組む時間が不足。

**解決策:**
* **直感的な教示システム:** ロボットのアームを手で動かして軌道を記録・再生する機能を実装。
* **成果:** プログラムが書けないメンバーでも演出作成が可能になり、開発効率を向上。「本田技研工業特別賞」を受賞した。
* [詳細はこちら](./Season2020)

### 3. モータドライバの設計と最適化 (MotorDriver_Projects)
**課題:** チーム内での回路のメンテナンス性向上と、通信ノイズによる誤動作の防止。

**解決策:**
* **制御方式:** マイコン1基につき1つのドライバを制御する方式でドライバ内のスイッチを使用し、動作確認を行えるように改良。
* **通信方式とツールの移行:** 通信方式をSPIからノイズに強いCAN通信へ変更し、設計ツールもEAGLEからKiCadへと移行して最適化を図った。
* [詳細はこちら](./MotorDriver_Projects)
