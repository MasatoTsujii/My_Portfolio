# Season2020: Hardware

## 概要
Season2020 ロボットで使用した回路基板の設計データ。
すべてのプロジェクトは **EAGLE** を使用して設計している。
閲覧用として、各プロジェクト内に回路図 (`Schematic.pdf`) および基板図 (`PCB.pdf`) を同梱している。

## ディレクトリ構成と基板説明

### 1. NucleoMB_Season2020
**概要:** ロボットのメイン制御ボード（マザーボード）
* **役割:** メインマイコンである **Nucleo-64 (STM32F446RE)** を搭載し、各センサやアクチュエータへのアクセスピンを使いやすい配置に引き出すための基板。

### 2. DirectTeaching_Controller
**概要:** ダイレクトティーチング操作用コントローラ
* **役割:** ロボットのアームに「動き」を教え込ませる際に使用する基板。
* **機能:**
    * ティーチングモードの切り替え
    * 位置データの保存番号指定
    * 現在位置データの取得および再生の実行

### 3. Dynamixel_TTL
**概要:** 通信変換基板
* **役割:** **Dynamixel AX-12A** とマイコン間の通信を仲介している。
* **機能:** AX-12Aは半二重の非同期シリアル通信（TTLレベル）を採用している。マイコン（UART）からの送受信信号を適切なタイミングで切り替えるためのICを搭載し、双方向通信を実現した。
