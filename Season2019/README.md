## 2. 競技ルールと課題
**【テーマ：らん♪ RUN Laundry】**

ロボットが「洗濯物」に見立てたTシャツ・バスタオル・シーツを、フィールド上の3本の物干し竿にきれいに干していく競技。
本大会は2台のロボット（1台は必ず自動ロボット）が協調してタスクを行うことが求められた。

**【競技の流れ】**
1.  ロボットが洗濯物（Tシャツ、バスタオル、シーツ）を回収する。
2.  Tシャツはハンガーにかけ、バスタオルとシーツは直接竿に干す。
3.  3本の竿の指定されたエリアに干すことで得点となり、干された状態の「美しさ」も評価対象となる。
