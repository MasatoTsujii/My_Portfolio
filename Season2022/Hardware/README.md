# Season2022: Hardware

## 概要
Season2022の自作コントローラーおよび周辺基板の設計データである。
設計には **KiCad** を使用した。
閲覧用として、回路図 (`.pdf`)、基板図 (`.pdf`)、および3Dレンダリング画像 (`.png`) をディレクトリ直下に配置している。

## ファイル構成
本プロジェクトでは、プロジェクトフォルダ階層ではなく、閲覧用ファイルを直下に配置する構成をとっている。

* `Controller2022_Schematic.pdf`: 回路図
* `Controller2022_PCB.pdf`: 基板レイアウト図
* `Controller2022_PCB_3D.png`: 基板の3D完成予想図

## プロジェクト詳細

### 1. Controller2022_ver0.1
**概要:** コントローラーのメイン基板
* **役割:** 本システムの核となる基板。メインマイコンとして **Nucleo-64 (STM32F446RE)** を搭載し、入力デバイスの信号処理および無線モジュールへのデータ受け渡しを行う。
* **インターフェース:**
    * 入力: タクトスイッチ、トグルスイッチ、ジョイスティック、ロータリーエンコーダ、ボリューム
    * 出力: 状態表示用LED、情報表示用ディスプレイ
    * 通信: 各種通信用基板との接続ピン

### 2. IM920_Conversion2
**概要:** 920MHz無線モジュール用変換基板
* **役割:** 無線モジュール (IM920sL) のピンアサインを、メイン基板と接続しやすい形に変換・中継するための基板。データ送信に必要なピンのみを引き出している。

### 3. JoyStick
**概要:** ジョイスティック用変換基板
* **役割:** ジョイスティックモジュールの配線を整理し、メイン基板への接続を容易にするために使用した。
