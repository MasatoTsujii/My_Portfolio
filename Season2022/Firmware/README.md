# Season2022: Firmware

## 概要
Season2022の自作コントローラー制御用プログラム群である。
すべてのプロジェクトは **PlatformIO mbed** 環境のみで開発を行った。

## プロジェクト一覧

### 1. Controller2022_MainBoard
**概要:** 大会本番で使用したメインプログラム
スイッチやジョイスティックの入力を取得し、無線モジュールを通じてロボット本体へ指令を送信する。

**主要ファイル構成:**
* **`src/Controller.cpp`, `include/Controller.h`**
    * 入力処理の中核プログラム。
    * エンコーダ、各スイッチ、ジョイスティックの生データを取得し、通信パケットとして送信可能なフォーマットへ変換・パッキングを行う。
* **`src/CRotaryEncoder.cpp`, `include/CRotaryEncoder.h`**
    * ロータリーエンコーダのパルスを読み取り、回転量を取得するためのクラス。
* **`src/IM920s.cpp`, `include/IM920s.h`**
    * 920MHz無線モジュール (IM920sL) を制御するためのプログラム。
    * モジュールの設定コマンドや送受信処理を一式実装している。

---

### 2. setting_IM920s
**概要:** 無線モジュール開発・検証用プログラム
**目的:** IM920sLモジュールの動作確認および、メイン基板 (`Controller2022_MainBoard`) への実装に向けた検証に使用した。コマンドの送受信テストなどを含む。

---

### 3. F303k8_Display
**概要:** ディスプレイ動作確認用プログラム
**目的:** ILI9341搭載 2.8インチTFT液晶（タッチパネル付）の表示テストに使用した。
**備考:** ライブラリは外部から持ってきた。
* 外部ライブラリをインポートしてSPI制御による描画を確認した。
* 本番機では情報の表示のみを行い、タッチパネル機能は使用していない。
