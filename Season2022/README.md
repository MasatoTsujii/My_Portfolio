# 高専ロボコン2022: 

## 1. 概要
**第35回 アイデア対決・全国高等専門学校ロボットコンテスト2022** に向けて制作した、紙飛行機射出ロボットの**自作コントローラー**。

3年ぶりの現地開催となった本大会において、私は5年生として1年生の技術指導を行いつつ、**専用コントローラーの開発**を担当した。

* **チーム:** 北九州工業高等専門学校 Bチーム
* **大会期間:** 2022年4月22日～2022年10月19日
* **制作期間:** 約3週間（コントローラー開発期間）

## 2. 競技ルールと課題
**【テーマ：ミラクル☆フライ ～空へ舞いあがれ！～】**

自作した紙飛行機をロボットが飛ばし、フィールド上の「スポット」「滑走路」「ベース」と呼ばれるエリアに着陸させて得点を競う対戦型競技。
3年ぶりの対面開催であり、対決での対応力が求められる大会であった。

## 3. コンセプト・技術

ロボットには**10基の射出砲塔**を搭載した。それぞれの砲塔の射出タイミングを独立して制御できる。

### 直面した技術的課題
1.  **操作ボタンの不足:** 10基の砲塔を独立制御（発射速度の調整や発射トリガー）し、かつ足回りを操作するには、市販のゲームパッド（PS4等）ではボタン数が圧倒的に不足していた。
2.  **通信環境の懸念:** 2019年の過去の現地開催において、2.4GHz帯（Bluetooth等）は会場内のWi-Fiやスマートフォンの電波干渉を受けやすく、通信トラブルが発生していた。
3.  **短納期:** 大会1ヶ月前に上記の問題が発覚し、実質3週間で解決策を形にする必要があった。

## 4. 技術的解決策・システム構成

上記の課題に対し、**「専用コントローラーの自作」** と **「通信方式の変更」** を行った。

### 1. 920MHz帯 自作コントローラーの開発
* **通信の安定化:**
    混信の激しい2.4GHz帯を避け、920MHz帯の無線モジュールを採用した。これにより、大会当日は一度も通信エラーを起こすことなく安定動作を実現した。また、不測の事態に備えBluetoothへの切り替えも可能な設計とした。
* **開発プロセス:**
    3週間という短期間で完成させるため、以下の工程で開発した。
    1.  **試作機の優先:** 必要最低限の機能を実装した試作機を最速で組み上げ、操縦者が練習できる時間を確保した。
    2.  **フィードバックによる改善:** 操縦者の意見（ボタン配置や操作感）を取り入れ、筐体設計者と連携して本番機を完成させた。
    3.  **プロトコルの流用:** 過去の通信プロトコルをベースに改良を加えることで、ファームウェア開発の工数を大幅に短縮した。

## 5. 担当箇所・技術詳細

### ハードウェア (Hardware)
* **使用CAD:** KiCad
* **自作コントローラー:**
    * 通信: 920MHz (IM920sL) / Bluetooth (予備)
    * 入力: ボタン、アナログスティック、エンコーダ

### ソフトウェア (Firmware)
* **言語:** C++
* **工夫点:**
    * 制御担当者と「電文表」を共有し、送受信側のプログラムを並行開発することで待ち時間を削減した。
